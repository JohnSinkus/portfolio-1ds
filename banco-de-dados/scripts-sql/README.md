# Scripts SQL

Este diretório contém uma coleção de scripts SQL desenvolvidos para diversas finalidades, incluindo criação de tabelas, inserção de dados, consultas e manipulação de bancos de dados.

## Conteúdo

Os scripts estão organizados por projetos ou atividades, abrangendo:

- **DDL (Data Definition Language)**: Comandos para criar, modificar e excluir estruturas de banco de dados (CREATE TABLE, ALTER TABLE, DROP TABLE).
- **DML (Data Manipulation Language)**: Comandos para inserir, atualizar e excluir dados (INSERT, UPDATE, DELETE).
- **DQL (Data Query Language)**: Comandos para consultar dados (SELECT).

## Como Utilizar

Os arquivos `.sql` podem ser executados em um cliente SQL ou IDE de banco de dados conectado a um SGBD (Sistema Gerenciador de Banco de Dados) como MySQL, PostgreSQL, SQL Server, Oracle, etc. Certifique-se de ter o banco de dados e as tabelas apropriadas criadas antes de executar os scripts de manipulação e consulta de dados.
