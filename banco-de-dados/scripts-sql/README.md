# Scripts SQL: Implementação e Manipulação de Bancos de Dados

Este diretório contém uma coleção abrangente de scripts SQL desenvolvidos para diversas finalidades, incluindo a criação de estruturas de banco de dados, inserção e manipulação de dados, e a execução de consultas complexas. Estes scripts refletem a aplicação prática da linguagem SQL em cenários de banco de dados.

## Conteúdo e Categorização

Os scripts estão organizados por projetos ou atividades, e abrangem as seguintes categorias da linguagem SQL:

*   **DDL (Data Definition Language)**: Comandos para definir, modificar e excluir estruturas de banco de dados, como `CREATE TABLE`, `ALTER TABLE` e `DROP TABLE`.
*   **DML (Data Manipulation Language)**: Comandos para gerenciar os dados dentro das tabelas, incluindo `INSERT`, `UPDATE` e `DELETE`.
*   **DQL (Data Query Language)**: Comandos para recuperar informações do banco de dados, principalmente através da instrução `SELECT`.

## Orientações para Execução

Os arquivos com extensão `.sql` podem ser executados em um cliente SQL ou em um ambiente de desenvolvimento integrado (IDE) de banco de dados, desde que estejam conectados a um Sistema Gerenciador de Banco de Dados (SGBD) relacional compatível (e.g., MySQL, PostgreSQL, SQL Server, Oracle). É fundamental assegurar que o ambiente do banco de dados esteja devidamente configurado e que as tabelas necessárias existam antes de executar scripts de manipulação e consulta de dados para evitar erros.
