# Exercícios Photoshop (PSD)

Este diretório contém arquivos de projeto do Adobe Photoshop (formato `.psd`) criados durante os exercícios práticos de Design Digital. Cada arquivo representa uma atividade ou um projeto específico, demonstrando habilidades em manipulação de imagens, composição e aplicação de efeitos.

## Conteúdo

Os arquivos `.psd` aqui presentes são resultados de atividades que exploram diversas funcionalidades do Photoshop, como:

- Edição e tratamento de imagens.
- Criação de composições visuais.
- Aplicação de filtros e ajustes.
- Uso de camadas e máscaras.

## Como Visualizar e Editar

Para visualizar e editar esses arquivos, é necessário ter o Adobe Photoshop instalado. Ao abrir um arquivo `.psd`, você terá acesso a todas as camadas e históricos de edição, permitindo entender o processo de criação e fazer modificações.
