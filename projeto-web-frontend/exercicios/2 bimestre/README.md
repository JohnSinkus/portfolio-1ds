# 2º Bimestre - Desenvolvimento Web Intermediário

Este diretório contém os exercícios e projetos desenvolvidos durante o segundo bimestre, aprofundando os conhecimentos em HTML e CSS, e explorando a criação de layouts responsivos e interfaces de usuário mais complexas.

## Conteúdo

- **`01.04 Aula/`**: Materiais e exemplos da aula do dia 01 de abril.
- **`15.04/`**: Atividades e projetos relacionados à data de 15 de abril.
- **`Atividade final do segundo bimestre das pizzas/`**: Projeto final do bimestre, provavelmente envolvendo um formulário ou sistema de pedidos de pizza.
- **`Chrome.html`**, **`Internet explorer.Html`**, **`MOZILLA-FIREFOX.html`**, **`Opera.html`**, **`SAFARI.HTML`**: Possíveis páginas HTML que demonstram compatibilidade ou diferenças entre navegadores.
- **`Dispositivos movei/`**: Conteúdo relacionado a desenvolvimento para dispositivos móveis ou responsividade.
- **`atividade dia 27.05.2025.zip`**: Arquivo compactado com atividades do dia 27 de maio de 2025.
- **`att dia 27 05/`**: Pasta com atividades do dia 27 de maio.
- **`blogueiro/`**: Projeto ou exercícios relacionados à criação de um blog.

## Tecnologias Utilizadas

- HTML5
- CSS3 (incluindo Flexbox e Grid)
- Design Responsivo

## Como Visualizar

Para visualizar os projetos, navegue até o diretório desejado e abra os arquivos `.html` em seu navegador web. Alguns projetos podem incluir arquivos `.css` para estilização.
