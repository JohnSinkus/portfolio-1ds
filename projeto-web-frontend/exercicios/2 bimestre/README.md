# 2º Bimestre: Desenvolvimento Web Intermediário e Responsivo

Este diretório compila os exercícios e projetos desenvolvidos durante o segundo bimestre, aprofundando os conhecimentos em HTML e CSS, e explorando a criação de layouts responsivos e interfaces de usuário mais complexas.

## Conteúdo Programático e Projetos

Os recursos aqui presentes abrangem:

*   **`01.04 Aula/`**: Materiais didáticos e exemplos práticos explorados durante a aula de 01 de abril.
*   **`15.04/`**: Atividades e projetos relacionados à data de 15 de abril, focados em conceitos intermediários de desenvolvimento web.
*   **`Atividade final do segundo bimestre das pizzas/`**: Projeto final do bimestre, possivelmente envolvendo a criação de um formulário ou sistema de pedidos de pizza, demonstrando a aplicação de conhecimentos em HTML e CSS.
*   **`Chrome.html`**, **`Internet explorer.Html`**, **`MOZILLA-FIREFOX.html`**, **`Opera.html`**, **`SAFARI.HTML`**: Páginas HTML que podem demonstrar compatibilidade ou diferenças de renderização entre diferentes navegadores, ou a criação de interfaces específicas para cada um.
*   **`Dispositivos movei/`**: Conteúdo relacionado ao desenvolvimento para dispositivos móveis ou à implementação de design responsivo.
*   **`atividade dia 27.05.2025.zip`**: Arquivo compactado contendo atividades desenvolvidas no dia 27 de maio de 2025.
*   **`att dia 27 05/`**: Pasta com atividades e projetos do dia 27 de maio.
*   **`blogueiro/`**: Projeto ou conjunto de exercícios relacionados à criação de um layout de blog, explorando estruturas e estilização.

## Tecnologias Dominadas

*   **HTML5**: Para a estruturação semântica de páginas web.
*   **CSS3**: Incluindo técnicas avançadas de estilização, Flexbox e Grid para layouts responsivos.
*   **Design Responsivo**: Adaptação de interfaces para diferentes tamanhos de tela e dispositivos.

## Instruções de Visualização

Para visualizar os projetos, navegue até o subdiretório desejado e abra os arquivos com extensão `.html` em seu navegador web. É fundamental que todos os arquivos de estilo (`.css`) estejam corretamente vinculados para a exibição adequada das interfaces.
